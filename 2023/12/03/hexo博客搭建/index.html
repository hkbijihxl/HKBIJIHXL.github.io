Hexo博客搭建

前期准备

  1. 提前安装好(node.js)[https://nodejs.org/en] 左边的那个
  2. 提前安装好(GIt)
  3. 提前注册好一个github账号

搭建

1.在本地找个目录放博客源文件，路径不要有中文

2.在选定的目录中，右键Git Bash Here

3.使用npm命令安装Hexo，输入：

​ npm install -g hexo-cli
​ hexo init blog
​ cd blog
​ hexo g

4.检查网站，在上一个Git Bash Here输入三条命令：

​ hexo n test_my_site #创建一篇博客文章

​ hexo g #生成

​ hexo s #预览

​ 之后就可以访问http://localhost:4000/预览blog了

5.配置ssh

​ (1):cd ~/.ssh #检查本机已存在的ssh密钥

​ 提示 No such file or directory 说明你是第一次使用git。

​ (2):ssh-keygen -t rsa -C “邮件地址”

​ 然后连续3次回车，最终会生成一个文件在用户目录下，打开用户目录，找到.ssh\id_rsa.pub文件，记事本 打开并复制里面的内容，打开你的github主页，进 入个人设置 -> SSH and GPG keys -> New SSH key：

​ (3):key填写”C:\Users\86178.ssh\id_rsa.pub”中的内容

​ (4):ssh连接，测试是否成功

​ ssh -T git@github.com # 注意邮箱地址不用改

​ (5):如果提示Are you sure you want to continue connecting (yes/no)?，输入yes，然后会看到下面这个，就算成功：

​ Hi liuxianan! You’ve successfully authenticated, but GitHub does not provide shell access.

​ (6):在blog目录下打开cmd(管理员身份)，输入：

​ npm install hexo-deployer-git –save

​ (7):Git Bash Here 中继续配置

​ git config –global user.name “luozhixiaowo” #你的github用户名
​ git config –global user.email “xxx@qq.com“ # 填写你的邮箱

6.代码上传

​ (1):在github上新建一个存储库，命名:github的用户名.github.io

​ (2):编辑_config.yml文件，在最下面更改，注意格式，缩进不对会导致上传不成功

​ (3):上传

​ hexo d
