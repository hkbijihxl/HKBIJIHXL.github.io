WireShark、AhMyth使用

2023-12-10 Comments

WireShark

一、WireShark软件安装

http://192.168.2.6:5244/d/%E7%BD%91%E7%9B%98/%E7%99%BE%E5%BA%A6%E7%BD%91%E7%9B%98/%E8%BD%AF%E4%BB%B6%E5%A4%A7%E5%85%A8/Wireshark-win64-3.0.0%E4%B8%AD%E6%96%87%E7%89%88.exe

二、Wireshark抓包

1、打开Wireshark，主界面如下

2、使用ping命令进行抓包

3、操作完成后相关数据包就抓取到了。为避免其他无用的数据包影响分析，可以通过在过滤栏设置过滤条件进行数据包列表过滤，icmp（只显示icmp协议的数据包）

三、wireshark抓包过滤器语法

（1）协议过滤

比较简单，直接在抓包过滤框中直接输入协议名即可。
TCP，只显示TCP协议的数据包列表
HTTP，只查看HTTP协议的数据包列表
ICMP，只显示ICMP协议的数据包列表

（2）IP过滤

host 192.168.2.18
src host 192.168.2.18
dst host 192.168.2.18

（3）端口过滤

port 80
src port 80
dst port 80

（4）逻辑运算符&& 与、|| 或、！非

src host 192.168.2.18 && dst port 80 抓取主机地址为192.168.2.80、目的端口为80的数据包
host 192.168.2.18 || host 192.168.2.12 抓取主机为192.168.2.18或者192.168.2.12的数据包
！broadcast 不抓取广播数据包

AhMyth

一、下载

http://192.168.2.6:5244/d/%E7%BD%91%E7%9B%98/%E7%99%BE%E5%BA%A6%E7%BD%91%E7%9B%98/%E8%BD%AF%E4%BB%B6%E5%A4%A7%E5%85%A8/%E6%B8%97%E9%80%8F%E5%B7%A5%E5%85%B7/%E8%BF%9C%E6%8E%A7/%E5%AE%89%E5%8D%93%E7%97%85%E6%AF%92%E7%9B%91%E5%90%AC/AhMyth_Win64.exe

二、ahmyth使用

1.安装打开后的界面是这样的， Victims是用来监听的界面。

2.APK Builder是用来生成后门程序安装包的界面。

3.在Source IP中添加自己的本机IP地址192.168.2.18，然后在Source Port添加一个任意不占用的端口地址，默认为42474，点击Build生成apk文件，根据显示文件位置找到安装包。
