Cent OS 7 命令

1.ls命令：

​ ls 查询当前目录

​ ls / 列出根目录(/)下的所有目录

​ ls -a 显示所有文件，包括隐藏文件(. 开头)

/bin 二进制文件，系统常规命令
/boot 系统启动分区，系统启动时读取的文件
/dev 设备文件
/etc 大多数配置文件
/home 普通用户的家目录
/lib 32位函数库
/lib64 64位库
/media 手动临时挂载点
/mnt 手动临时挂载点
/opt 第三方软件安装位置
/proc 进程信息及硬件信息
/root 临时设备的默认挂载点
/sbin 系统管理命令
/srv 数据
/var 处理动态数据
/sys 内核相关信息
/tmp 临时文件
/usr 用户相关设定

​ ls -al 显示路径中所有文件的详细信息(属性)
-普通文件 d目录文件
所有者 所属组 其他人(3个一组)
u g o
文件数量 文件大小
rwx 读写执行 最大权限7
421
. 当前目录
.. 上一级目录
​ ls -d 查看目录

2.touch命令:

​ touch 创建文件

​ 只有可执行、文本文件两种 Linux中

3.mkdir命令:

​ mkdir 创建目录
​ mkdir -p 递归创建

4.cd命令:

​ cd 切换目录
​ cd . cd .. cd /
​ cd / //切换到根目录
​ cd /bin //切换到根目录下的bin目录
​ cd ../ //切换到上一级目录 或者使用命令：cd ..
​ cd ~ //切换到home目录
​ cd - //切换到上次访问的目录
​ cd xx(文件夹名) //切换到本目录下的名为xx的文件目录，如果目录不存在报错
​ cd /xxx/xx/x //可以输入完整的路径，直接切换到目标目录，输入过程中使用tab键快速补全

5.whoami命令：

 whoami	显示当前用户名

6.id命令:

​ id 查看用户基本信息

7.pwd命令:

​ pwd 当前目录

8.rm命令:

​ rm 删除
​ rm -f 强制删除
​ rm -rf 强制递归删除

9.mv命令:

​ mv 移动文档

10.管道|grep

​ ls / |grep t 查询根目录下包含”t”字母的文档名称
